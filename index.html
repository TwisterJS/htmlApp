TwisterJS: Teaching kids to code

This project is managed by Carmalou. You can see her GitHub page here. The goal of this project is to expose under-privileged young children to technology. This is being done through apps which will be available to download through the App Store and Google Play.

You can see what we are working on here.

Everything created here is available for use by anyone -- provided they do not charge for it, or use the code in another project which will cost money. The goal of TwisterJS is to give opportunity to children who not otherwise have this opportunity. As such, everything here is open-source.

Please use it to help those in your community.
